About Me

Hello, I'm Yan Cheng

Portfolio

Land Cover Classification

Flood Risk Assessment

Created a comprehensive flood risk assessment tool using DEM analysis and hydrological modeling. The project helped local authorities identify high-risk areas and develop evacuation strategies.

Urban Heat Island Analysis

Analyzed urban heat island effects using thermal infrared imagery and meteorological data. Provided recommendations for green infrastructure to mitigate temperature extremes in city centers.

Deforestation Monitoring

Implemented a real-time deforestation monitoring system using Google Earth Engine. The system provides automated alerts when forest loss is detected in protected areas.

Crop Yield Prediction

Developed a predictive model for crop yield estimation using multi-temporal satellite data and weather information. Helped farmers optimize their resource allocation and improve productivity.

Coastal Erosion Study

Conducted a long-term coastal erosion analysis using historical satellite imagery and LiDAR data. Quantified erosion rates and identified vulnerable coastal segments for protection measures.
